# Список чипсов

Тестовое задание: компонент однострочного списка чипсов с попапом для элементов, которые не поместились, выбором по клику и адаптацией к ширине контейнера.

## Демо

[Смотреть демо](https://dkskv.github.io/chips/?path=/story/ui-chipsoverflowrow--default&globals=viewport.value:desktop)

Публикуется на GitHub Pages при push в `master`.

## Запуск

```bash
npm install
npm run storybook
```

Тесты:

```bash
npm test
```
