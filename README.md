# Список чипсов

Тестовое задание: компонент однострочного списка чипсов с попапом для элементов, которые не поместились, выбором по клику и адаптацией к ширине контейнера.

## Демо

Storybook: [https://dkskv.github.io/chips/](https://dkskv.github.io/chips/)

Публикуется на GitHub Pages при push в `master`.

## Запуск

```bash
npm install
```

Демо компонентов локально:

```bash
npm run storybook
```

Откроется [http://localhost:6006](http://localhost:6006).

Тесты:

```bash
npm test
```
